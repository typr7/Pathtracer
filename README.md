# Pathtacer
This is a pathtracer that written by C++20, it uses NVIDIA's OptiX as its graphics API.
# Feature
- Monte Carlo Path Tracing
- PBR Material
- Metalness Workflow
- Simple Arcball Camera
- Maybe More
